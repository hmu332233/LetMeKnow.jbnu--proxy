## 알려줘전북대 proxy

알려줘전북대로 들어오는 메세지를 core로 넘겨주는 역할을 하는 앱.

크게 다음과 같은 역할을 한다.

- request를 분석하여 원하는 앱으로 넘겨주기
- request를 db에 기록
- 기능의 on/off
- 특정 유저의 block
